## Apa itu Next.js?

Next.js adalah kerangka kerja React untuk membangun aplikasi web full-stack. Anda menggunakan Komponen React untuk membangun antarmuka pengguna, dan Next.js untuk fitur-fitur tambahan dan optimalisasi.

Di dalamnya, Next.js juga mengabstraksi dan secara otomatis mengonfigurasi perangkat yang dibutuhkan untuk React, seperti penggabungan, kompilasi, dan lainnya. Ini memungkinkan Anda fokus pada pembuatan aplikasi Anda daripada menghabiskan waktu dengan konfigurasi.

Baik Anda seorang pengembang individu atau bagian dari tim yang lebih besar, Next.js dapat membantu Anda membangun aplikasi React yang interaktif, dinamis, dan cepat.

## Fitur Utama

Beberapa fitur utama Next.js meliputi:

| Fitur                                                                    | Deskripsi                                                                                                                                                                                                |
| ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Routing](https://nextjs.org/docs/app/building-your-application/routing) | Sebuah router berbasis sistem file yang dibangun di atas Komponen Server yang mendukung tata letak, routing bersarang, keadaan pemuatan, penanganan kesalahan, dan banyak fitur lainnya.                 |
| [Rendering]()                                                            | Rendering di Sisi Klien dan Sisi Server dengan Komponen Klien dan Server. Ditingkatkan lebih lanjut dengan Rendering Statis dan Dinamis di server dengan Next.js. Streaming di Edge dan runtime Node.js. |
| [Data Fetching]()                                                        | Pengambilan data yang disederhanakan dengan `async`/`await` pada Komponen Server, dan API pengambilan diperluas untuk memoisasi permintaan, penyimpanan data, dan validasi ulang data.                   |
| [Styling]()                                                              | Dukungan untuk metode styling pilihan Anda, termasuk Modul CSS, Tailwind CSS, dan CSS-in-JS.                                                                                                             |
| [Optimizations](g)                                                       | Optimisasi gambar, font, dan skrip untuk meningkatkan Core Web Vitals dan Pengalaman Pengguna aplikasi Anda.                                                                                             |
| [TypeScript]()                                                           | Dukungan yang ditingkatkan untuk TypeScript, dengan pemeriksaan tipe yang lebih baik dan kompilasi yang lebih efisien, serta plugin TypeScript kustom dan pemeriksa tipe.                                |

## Cara menggunakan dokumentasi Ini

Di sisi kiri layar, Anda akan menemukan bilah navigasi dokumen. Halaman-halaman dokumen disusun secara berurutan, dari dasar hingga tingkat lanjut, sehingga Anda dapat mengikuti langkah-langkahnya secara bertahap saat membangun aplikasi Anda. Namun, Anda dapat membacanya dalam urutan apa pun atau melompat ke halaman-halaman yang relevan dengan kasus penggunaan Anda.

Di sisi kanan layar, Anda akan melihat tabel isi yang memudahkan navigasi antara bagian-bagian halaman. Jika Anda perlu dengan cepat menemukan sebuah halaman, Anda dapat menggunakan bilah pencarian di bagian atas atau pintasan pencarian `Ctrl+K` atau `Cmd+K`.

Untuk memulai, periksa panduan [Instalasi]().
